# 🖌️ Drawing App

A freehand drawing app built with React, TypeScript, Canvas API and Zustand.

## ✨ Features
- Freehand drawing on canvas
- 8 preset colors + custom color picker
- Adjustable brush size
- Eraser tool
- Undo support
- Download drawing as PNG
- Clear canvas

## 🛠️ Tech Stack
- React 19
- TypeScript
- Tailwind CSS v4
- Zustand (global state)
- Canvas API (native browser)
- Vite

## 🚀 Run Locally
```bash
git clone https://github.com/your-username/drawing-app.git
cd drawing-app
npm install
npm run dev
```

- useRef — DOM access without re-renders
